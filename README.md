Проект №3: Finance Tracker
Програмата работи чрез меню с команди:
setup - инициализира нов профил, като пита потребителя за брой месеци, които иска да бъдат отчетени
add - добавя данни за приход и разход за избран месец (Jan, Feb...)
report - извежда таблица със всички въведени месеци и техните приходи, разходи, баланс
search - дава детайлна информация за конкретен месец, в отношение на разходи/приходи
sort - сортира данните и показва Топ 3 месеца според  избрания критерий (income, expense, balance)
forecast - дава прогноза за бъдещи спестявания на базата на средния месечен баланс до момента
chart - визуализира приходите на избраните месеци
exit - приключва приложението
